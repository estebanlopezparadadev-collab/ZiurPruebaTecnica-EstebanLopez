# Prueba Técnica: Desarrollador .NET - Ziur Software

Esta solución ha sido desarrollada para el proceso de selección de **Ziur Software**, enfocándose en la creación de una interfaz moderna y eficiente para el consumo de servicios externos utilizando las últimas tecnologías de Microsoft.

## Tecnologías Utilizadas
* **Framework:** .NET 8.0 (Blazor Web App).
* **Arquitectura:** Patrón de Servicio con Inyección de Dependencias.
* **UI/UX:** Bootstrap 5, Bootstrap Icons y CSS Isolation.
* **Lógica:** Consultas LINQ para filtrado dinámico en memoria.

## Características Implementadas
1. **Consumo de API REST:** Implementación de un `DataService` robusto que gestiona la autenticación mediante Bearer Token y la recuperación de documentos.
2. **Filtrado en Tiempo Real:** Sistema de búsqueda optimizado que filtra la tabla de documentos automáticamente mientras el usuario escribe, mejorando la experiencia de usuario (UX).
3. **Interfaz Profesional:** Diseño limpio con estados de carga (Spinners), manejo de errores y visualización clara de estados (Activo/Inactivo) mediante Badges.
4. **Diseño Responsivo:** Adaptabilidad total a diferentes tamaños de pantalla.
